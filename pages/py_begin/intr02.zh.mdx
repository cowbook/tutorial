# B2 安装Python

## 0.前言

本篇将介绍如何安装 `Python官方` 提供的运行环境，目前Python稳定的版本是3.13。
官方Python解释器及环境，可以说是简单而粗糙，并不适合专业程序员编写Python应用。

不仅是Python，任何完整的应用，源程序都是由很多个文件组成的。
所以，专业程序员需要专业的IDE工具，比如VsCode、PyCharm这样的工具，这些工具提供项目管理、格式化文本、自动填充等功能。

安装过程很简单，所以咱先掌握这个，后面会教授其他高级IDE的安装。

## 3.IDLE环境验证

很多小哥哥安装到此就结束了，在此应该进行如下2个功能的验证，如果不成功那么说明您前面的程序有问题，请重新卸载，按照本文勾选好，再来安装一次。

点击开始菜单，启动Python3.8下面的IDLE。 IDLE是Python最基本的命令行，提供简单的程序编辑和命令行功能，基本上我们今后用不到它。

![启动IDLE](/images/intr02.zh_2024-04-06-19-48-23.png)
图4 启动原生编辑器IDLE

值得一提，菜单中的 Python 3.8 Manuals ，是Python的语言指南，是学习Python真正的好东东，如果英文在小学6年级之上，建议好好阅读相关章节。

下图是打开IDLE后的界面，`>>>`这三个大于号命令行的提示符，提示我们已经进入了python的运行环境。

![](/images/intr02.zh_2024-04-06-20-09-08.png)

验证成功后，我们再来验证在命令行下如何运行Python的源程序。

## 4.命令行环境运行Python
通过如下的步骤打开Windows的命令行环境，右击 开始菜单 -> “命令提示符（管理员）”，可以打开如下的命令行，为了方便今后运行程序，可右击开始菜单。

![](/images/intr02.zh_2024-04-06-20-10-00.png)

在命令行中，如下图，输入下面的命令行，这是终端拷贝命令，用来把con（就是控制台）copy到文件test.py文件。讲人话说就是，新建test.py文件。

```bash
copy con test.py           
```

输入命令回车，DOS命令行提示符会消失进入输入状态，输入如下的python程序：

```python
print('Hello World!') 
```

输完后按下回车，再按下键盘上的 Ctrl+Z 键（表示文件尾），然后再按下回车，保存文件，回到命令行。

现在可以通过在命令行，直接输入这个.py的文件名，来运行验证。

![](/images/intr02.zh_2024-04-06-20-11-40.png)


如上图，我们创建了Python的程序后，直接在命令行运行它，在命令行中直接输入这个文件名称test.py如果你看到返回了程序运行的结果Hello World!说明验证成功，如果有问题，那么重新安装。
