# B2 安装Python

## 0.前言

本篇将介绍如何安装 `Python官方` 提供的运行环境，目前Python稳定的版本是3.13。
官方Python解释器及环境可以说是简单而粗糙，并不适合专业程序员编写Python应用。

不仅是Python，任何完整的应用，源程序都是由很多个文件组成的。
所以，专业程序员需要专业的IDE工具，比如VsCode、PyCharm这样的工具，这些工具提供项目管理、格式化文本、自动填充等功能。

因为这个安装这个简单，所以先学这个，后面会介绍其他的安装方式。

## 1.官方Python的网站

## 3.IDLE环境验证

很多小哥哥安装到此就结束了，在此应该进行如下2个功能的验证，如果不成功那么说明您前面的程序有问题，请重新卸载，按照本文勾选好，再来安装一次。

点击开始菜单，启动Python3.8下面的IDLE。 IDLE是Python最基本的命令行，提供简单的程序编辑和命令行功能，基本上我们今后用不到它。

## 4.命令行环境运行Python
通过如下的步骤打开Windows的命令行环境，右击 开始菜单 -> “命令提示符（管理员）”，可以打开如下的命令行，为了方便今后运行程序，我们 右击开始菜单。

![](/images/intr02.zh_2024-04-06-20-10-00.png)

在命令行中，如下图，输入下面的命令，这是一个拷贝命令，用来把con就是控制台copy到文件test.py文件。用人话说就是新建一个test.py文件。

```bash
copy con test.py           
```

输入这个命令回车后，DOS命令行提示符会消失，输入如下的python程序：

```python
print('Hello World!') 
```

输完后按下回车，再按下键盘上的 Ctrl+Z 键，然后再按下回车，就可以保存文件。
